var:
Var is a data type and it has funtion scope. That means if we declare a variable using the 'var' data type then it can accesable inside that specific funtion where the variable is declared.

let:
Let is also a data type but it has block scope. That means it can accesable from that block where it has beed declared.

const:
Finally JS also has a data type called const and it also has block scope.

Null:
Null means an empty value. That's mean there are nothing or empty.

Undefined:
Undefined means the variable has beed defined but the value has not been assigned.

REST API
API means application programming interface. And REST API is that technology which define that how applications can communicate with each other.
